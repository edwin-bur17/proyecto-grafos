Información del Proyecto

Sistema de Guía de Compras Inteligente

Objetivo

Facilitar al usuario la realización de compras en el menor tiempo posible dentro de entornos desconocidos, sugiriendo la ruta óptima para localizar cada producto considerando factores como niveles de congestión.

Estructuras de Datos

1. Lista Enlazada

Gestiona los productos que el usuario necesita adquirir. Cada producto es un nodo conectado al siguiente mediante punteros.

Operaciones: agregar, eliminar, buscar, recorrer

2. Árbol de Categorías

Representa la organización jerárquica de categorías y productos en la tienda. Permite búsqueda eficiente y navegación por categorías.

Operaciones: buscar, obtener productos, recorrer árbol

3. Grafo de Rutas

Modela las conexiones entre pasillos con pesos dinámicos basados en distancia y congestión. Utiliza el algoritmo de Dijkstra para encontrar rutas óptimas.

Operaciones: ruta más corta, optimizar múltiples destinos, actualizar congestión

Integración

El sistema integra las tres estructuras de datos en un proceso unificado:

  1. El usuario agrega productos a su lista de compras (Lista Enlazada)
  2. El sistema busca cada producto en el árbol de categorías
  3. Se identifican los pasillos donde se encuentran los productos
  4. El grafo calcula la ruta óptima para visitar todos los pasillos
  5. Se considera el nivel de congestión para optimizar el tiempo

Tecnologías

  • Backend: Django 5.2.8 (Python)
  • Frontend: HTML5, CSS3
  • Base de Datos: SQLite
  • Estructuras de Datos: Implementación manual en Python
